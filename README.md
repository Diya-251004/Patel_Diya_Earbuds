# Claro Earbuds AR Model Project

## Submission 1 - AR Model with Interactive Hotspots

### Requirements Completed
- Exported AR object from Cinema 4D (MMED-1058) integrated
- CSS3, JavaScript, and GreenSock library for hotspot animations
- Array/object data storage for hotspots
- Images included for each hotspot

### Setup Instructions

**Compile SCSS to CSS**
- Compile `main.scss` to `css/main.css`
- Use SCSS compiler or VS Code extension

### Features
- Model-viewer integration for AR/3D viewing
- GSAP animations for hotspot entrance and interactions
- Responsive design
- Interactive hotspot annotations with images
